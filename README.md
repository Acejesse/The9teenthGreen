# The9teenthGreen

The9teenthGreen is a mobile Android application built using Kotlin and Jetpack Compose. It is designed for golf enthusiasts to browse and view different golf equipment categories.

Features
Home screen with golf-themed UI
Navigation to product categories
Golf Clubs list (11 detailed items)
Golf Balls list (10 items)
Back navigation between screens
Clean Material3 design
Screens
MainActivity (Home Page)
ClubsActivity (Golf Clubs)
GolfBallsActivity (Golf Balls)
Tech Stack
Kotlin
Jetpack Compose
Material3
Android Studio
